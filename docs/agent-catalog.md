# Agent Catalog

## toolbox-navigator

Shows an ADHD-friendly menu and recommends the right Open Tools command.

Command: `/toolbox`

## github-completion-auditor

Read-only audit for working tree cleanliness, pushed state, deployment status, and local running jobs.

Command: `/audit-completion`

## session-kanban-keeper

Maintains `.opencode/session-kanban.md` so unfinished work survives token limits.

Commands: `/show-kanban`, `/save-kanban`

## repo-cartographer

Creates `.opencode/repo-map.md` for durable repo understanding.

Command: `/map-repo`

## pr-reviewer

Reviews diffs or PRs for bugs, regressions, missing tests, and risks.

Command: `/review-diff`

## release-captain

Runs release readiness preflight and returns go/no-go.

Command: `/release-check`

## security-privacy-sentinel

Checks for secrets, PII leaks, auth/session issues, unsafe logs, and privacy regressions.

Command: `/security-check`

## session-archivist

Writes dated session summaries under `.opencode/session-log/`.

Command: `/archive-session`

## decision-journalist

Writes lightweight ADRs under `.opencode/decisions/`.

Command: `/adr`

## founder-strategist

Creates founder-level decision memos with priorities, non-goals, success criteria, and risks.

Command: `/founder-memo`

## product-manager

Turns strategy or ambiguity into concrete product decisions, launch priorities, and acceptance criteria.

Command: `/product-decisions`

## conversion-copywriter

Writes conversion-focused copy without inventing proof, metrics, or testimonials.

Command: `/copy-review`

## seo-llm-discovery

Reviews public surfaces for SEO, metadata, schema, FAQ, internal links, and LLM discoverability.

Command: `/seo-review`

## ux-conversion-reviewer

Reviews UX hierarchy, trust, accessibility, responsive behavior, and conversion friction.

Command: `/ux-review`

## senior-frontend

Handles frontend implementation or review with accessibility, responsiveness, API handling, and verification focus.

Command: `/frontend-task`

## senior-backend

Handles backend implementation or review with API contracts, data safety, migrations, auth, jobs, and integrations focus.

Command: `/backend-task`

## qa-launch-verifier

Runs repeatable QA and launch checks for routes, forms, desktop/mobile behavior, builds, and smoke tests.

Command: `/qa-launch`

## railway-maintenance

Audits Railway service roots, build/start commands, health checks, required env names, and deployment blockers.

Command: `/railway-check`

## bounded-agent-architect

Designs bounded, reviewable AI workflows with strict contracts, validation, persistence, and audit trails.

Command: `/design-agent-workflow`

## structured-output-designer

Designs strict JSON output schemas, prompt rules, validation, persistence, and tests for AI tasks.

Command: `/design-structured-output`

## document-extraction-agent

Designs exact-text extraction workflows that preserve source order, blocks, and uncertainty.

Command: `/design-extraction`

## domain-classifier

Designs controlled-label classification agents with evidence status, confidence, reasons, and strict JSON output.

Command: `/design-classifier`
