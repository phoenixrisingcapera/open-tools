Agents
Specialized personas. Explicit boundaries.
Each agent has a defined purpose, permissions, and rules. No overlap. No ambiguity.

Session Memory

Never lose context across token limits. The session-kanban-keeper maintains a persistent todo board. The session-archivist writes daily summaries. The decision-journalist captures architecture decisions as lightweight ADRs.

Run /show-kanban at the start of every session. Run /save-kanban at the end. Your work state survives.

/show-kanban → Active: 3 tasks → In Review: 1 task → Blocked: 0 tasks /save-kanban → Saved 4 tasks to .opencode/session-kanban.md

Preflight Checks

Before you push, run the preflight pack. /audit-completion checks git state. /review-diff finds bugs and regressions. /security-check catches secret leaks. /release-check validates deployment readiness.

Every check is read-only. Nothing changes unless you ask it to.

$ /security-check → Scanning diff for secrets... → Scanning for PII exposure... → Checking auth/session changes... → Result: 0 findings $ /release-check → Ready to deploy

Feature Delivery

Build features without architecture drift. /fullstack-feature validates backend, frontend, contracts, and wiring. /build-feature-no-drift prevents duplicate routes and endpoints. /find-idle-surfaces discovers orphaned code.

Every feature is validated against the existing codebase before it's written.

$ /fullstack-feature → Verifying route: /api/users → Checking schema: user.model.ts → Validating API client: useUsers.ts → Checking UI state: UsersList.tsx → Result: All contracts aligned

Role-Based Agents

Dedicated agents for every role. /frontend-task handles UI correctness and accessibility. /backend-task covers API contracts and data safety. /qa-launch runs routes, forms, and smoke tests. /railway-check audits deployment health.

Each agent works within its domain. No bleed. No overlap.

$ /frontend-task → Checking responsive breakpoints → Validating ARIA attributes → Testing keyboard navigation → Result: 2 issues found $ /backend-task → All checks passed
